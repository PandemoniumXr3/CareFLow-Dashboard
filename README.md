🧡 CareFlow Dashboard

Een persoonlijk dashboard dat helpt met structuur, overzicht en minder mentale druk – zowel tijdens werk als na werk.


💡 Over dit project


Dit project heb ik gemaakt als cadeau voor mijn moeder, die werkt als verpleegkundig specialist.
Ze heeft lange en drukke werkdagen met veel:
verslagen schrijven
medicatie bijhouden
follow-ups regelen
beslissingen maken onder tijdsdruk
Daarnaast is het na werk vaak lastig om nog na te denken over dingen zoals eten of boodschappen.
Daarom heb ik CareFlow gebouwd:
een simpele tool die helpt om sneller te werken en makkelijker te schakelen naar rust.
Dit is geen medisch systeem, maar een persoonlijke workflow assistent.

⚙️ Wat kan het?
CareFlow bestaat uit twee delen:

🩺 Werkmodus
Helpt tijdens werk met: ->

✍️ Verslag generator

💊 Medicatie overzicht

📋 Follow-up lijst

🧠 Snelle zinnen (templates)

📊 Vandaag overzicht

Doel: minder typen, minder nadenken, meer overzicht.



🌙 Thuis / Avondmodus ->
Helpt na werk met:

🍽️ “Wat eten we?”

🛒 Boodschappen suggesties

👥 Aantal personen kiezen

🧊 Overzicht wat er nog in huis is

Doel: simpele keuzes maken als je moe bent.




⚡ Snelle shortcuts

Alles op één plek:

Notion openen
Mail openen
WhatsApp openen


🎨 Design

Het dashboard is ontworpen als een rustige “cockpit”:
clean en overzichtelijk
zachte kleuren + oranje accenten (persoonlijk)
afgeronde kaarten
weinig afleiding
Alles is gericht op rust en duidelijkheid.


🔐 Privacy

Dit project is bewust simpel en lokaal gehouden:
geen backend
geen database
geen login
geen externe opslag
Alle data blijft lokaal (eventueel via localStorage).
Het is niet bedoeld voor echte patiëntgegevens.


🛠️ Gebouwd met

HTML
CSS
JavaScript
Zonder frameworks, zodat het makkelijk aan te passen en uit te breiden is.


🚀 Waarom dit project?

Ik wilde iets maken dat:
echt nuttig is in het dagelijks leven
tijd bespaart
stress vermindert
en persoonlijk voelt
In plaats van alleen iets technisch te bouwen, heb ik geprobeerd een systeem te maken dat iemand echt gebruikt.
